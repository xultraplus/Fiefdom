# 《采邑春秋》UI交互设计

## 设计原则

1. **信息层级**：重要信息优先显示
2. **操作便捷**：常用功能一键到达
3. **视觉统一**：青铜器风格贯穿全游戏
4. **响应及时**：操作立即反馈
5. **符合史实**：春秋时期审美

---

```
┌─────────────────────────────────────────┐
│  [声望] [爵位]    采邑春秋    [金钱][粮食] │ ← 顶部栏
├─────────────────────────────────────────┤
│                                         │

│              游戏画面                    │
│          (TileMap + 角色)                │

│  [背包] [管理] [地图] [六艺] [睡眠]       │ ← 底部栏
└─────────────────────────────────────────┘
```

### 1.2 顶部信息栏

#### 左侧：玩家状态
```
┌────────────────┐
│ 📜 声望: 1250  │
│ 👑 爵位: 上士  │
│ ⏰ 第3年 夏至  │
└────────────────┘
```

#### 右侧：资源状态
```
┌────────────────┐
│ 💰 金钱: 1,250 │
│ 🌾 粮食: 580   │
│ 🛡️ 安全: 85%   │
└────────────────┘
```

**交互**：
- 鼠标悬停：显示详细信息
- 点击：打开对应面板

**颜色编码**：
- 正常状态：白色
- 警告状态（资源<20%）：黄色
- 危险状态（资源<10%）：红色闪烁

### 1.3 底部快捷栏

#### 按钮布局

| 按钮 | 图标 | 快捷键 | 功能 |
|------|------|--------|------|
| 背包 | 🎒 | I | 打开背包 |
| 管理 | 👥 | Tab | 打开门客管理 |
| 地图 | 🗺️ | M | 打开世界地图 |
| 六艺 | 📖 | L | 打开六艺面板 |
| 睡眠 | 🌙 | Space | 结束当前日 |

**视觉反馈**：
- 悬停：按钮放大110%
- 点击：按钮缩小90%然后恢复
- 有新通知：按钮右上角红点提示

### 1.4 快捷栏扩展

点击"更多"按钮展开：

| 按钮 | 快捷键 | 功能 |
|------|--------|------|
| 事件 | E | 查看当前事件 |
| 任务 | Q | 查看任务列表 |
| 设置 | Esc | 打开设置菜单 |
| 保存 | S | 快速保存 |

## 5. 反馈设计

### 5.1 浮动文字

**获得资源**：
```
+5 金钱 (金色上浮)
```

**消耗资源**：
```
-10 粮食 (红色下浮)
```

**经验获得**：
```
+15 XP (紫色上浮)
```

### 5.2 通知系统

#### 通知类型

| 类型 | 图标 | 颜色 | 持续时间 |
|------|------|------|----------|
| 信息 | ℹ️ | 白色 | 3秒 |
| 成功 | ✓ | 绿色 | 3秒 |
| 警告 | ⚠️ | 黄色 | 5秒 |
| 错误 | ✗ | 红色 | 5秒 |
| 解锁 | 🔓 | 紫色 | 5秒 |

#### 通知队列

- 最多同时显示3个
- 新通知从右侧滑入
- 旧通知向上移动或淡出

## 6. 可访问性

### 6.1 色盲模式

- 正常模式：红/绿/蓝
- 色盲模式：使用图标+文字，避免仅用颜色区分

### 6.2 字体大小

- 小：12px
- 中：14px（默认）
- 大：18px

### 6.3 快捷键自定义

- 所有快捷键可在设置中修改
- 支持游戏手柄
