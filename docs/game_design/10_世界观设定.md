# 《采邑春秋》世界观设定

## 核心概念

本作基于真实春秋历史（公元前770-476年），融合历史考据与游戏性。

---

## 1. 历史背景

### 1.1 时代背景

**时间设定**：春秋时期（公元前770-476年）

**历史特征**：
- 周王室衰微，诸侯争霸
- 井田制瓦解，初税亩兴起
- 诸子百家争鸣
- 礼崩乐坏与复古思潮并存

**社会结构**：
```
周天子（王）
  ├─ 诸侯（公、侯、伯、子、男）
  │   ├─ 卿大夫（卿、大夫）
  │   │   ├─ 士（上士、中士、下士）
  │   │   └─ 庶人
  │   └─ 平民
  └─ 奴隶
```

### 1.2 地理设定

**玩家封地位置**：中原地区（今河南、山东交界）

**周边势力**：
- **北**：晋国（霸主之一）
- **东**：齐国（经济强国）
- **南**：楚国（文化不同）
- **西**：秦国（后起之秀）
- **中央**：周王室（名义共主）

**重要城市**：
- 洛邑（王都，今洛阳）
- 曲阜（鲁国，孔子故乡）
- 临淄（齐国，商业中心）

---

## 2. 井田制历史

### 2.1 制度起源

**传说起源**：商朝，完善于西周

**基本形态**：
```
    井    井    井
  ┌─┬─┬─┐┌─┬─┬─┐┌─┬─┬─┐
  │私│私│私││私│私│私││私│私│私│
  ├─┼─┼─┤├─┼─┼─┤├─┼─┼─┤
  │私│公│私││私│公│私││私│公│私│
  ├─┼─┼─┤├─┼─┼─┤├─┼─┼─┤
  │私│私│私││私│私│私││私│私│私│
  └─┴─┴─┘└─┴─┴─┘└─┴─┴─┘
    井    井    井
```

**公私关系**：
- 公田：中间一格，产出归贵族
- 私田：周围八格，产出归农户

**制度目的**：
- "溥天之下，莫非王土"
- 维持等级秩序
- 确保贵族收入

### 2.2 瓦解过程

**历史原因**：
1. 生产力发展（铁器、牛耕）
2. 人口增长
3. 战争频繁
4. 贵族贪欲

**标志性事件**：
- 公元前594年：鲁国"初税亩"
- 公元前548年：楚国"量入修赋"
- 公元前538年：郑国"作丘赋"

**历史意义**：
- 井田制瓦解 → 封建制度建立
- 私有制兴起 → 商品经济繁荣
- 社会变革 → 诸子百家争鸣

## 3. 诸子百家

### 3.1 儒家

**代表人物**：孔子（孔丘）

**核心思想**：
- 仁：爱人
- 礼：规范
- 德：道德
- 治国：以德治国

**政治主张**：
- 恢复周礼
- 德治为主，刑罚为辅
- 选贤与能

**游戏表现**：
- 喜欢祭祀、典籍
- 厌恶法家政策
- 忠诚度高

### 3.2 墨家

**代表人物**：墨子（墨翟）

**核心思想**：
- 兼爱：平等爱人
- 非攻：反对侵略战争
- 节用：节约资源
- 尚贤：任人唯贤

**政治主张**：
- 兼爱非攻
- 节俭反奢
- 科技兴国

**游戏表现**：
- 擅长工巧
- 节俭
- 喜欢劳动

### 3.3 法家

**代表人物**：韩非子

**核心思想**：
- 法：法律
- 术：权术
- 势：威势

**政治主张**：
- 以法治国
- 君主专制
- 富国强兵

**游戏表现**：
- 严苛高效
- 喜欢秩序
- 厌恶传统

### 3.4 农家

**代表人物**：许行

**核心思想**：
- 君民并耕
- 重农抑商
- 平均主义

**政治主张**：
- 君主应与民同劳
- 重视农业
- 反对剥削

**游戏表现**：
- 擅长种田
- 喜欢土地
- 厌恶战争

### 3.5 兵家

**代表人物**：孙武（孙子）

**核心思想**：
- 战争理论
- 战略战术
- 将领素质

**政治主张**：
- 慎战
- 知彼知己
- 不战而屈人之兵

**游戏表现**：
- 擅长战斗
- 勇猛
- 喜欢训练

## 4. 周礼制度

### 4.1 五等爵位

| 爵位 | 天子封 | 诸侯封 | 义务 | 权利 |
|------|--------|--------|------|------|
| 公 | 王 | - | 朝觐、征伐 | 100里封地 |
| 侯 | 王 | - | 朝觐、征伐 | 70里封地 |
| 伯 | 王 | - | 朝觐、征伐 | 50里封地 |
| 子 | 王 | 诸侯 | 纳贡 | 50里封地 |
| 男 | 王 | 诸侯 | 纳贡 | 30里封地 |

### 4.2 等级秩序

**服饰制度**：
- 天子：衮服（十二章纹）
- 诸侯：鼹衣（九章纹）
- 卿大夫：玄衣纁裳（七章纹）
- 士：布衣（无纹）
- 庶人：粗布（褐色）

**乐舞制度**：
- 天子：八佾（64人）
- 诸侯：六佾（48人）
- 卿大夫：四佾（32人）
- 士：二佾（16人）

**祭祀制度**：
- 天子：郊祀（祭天）
- 诸侯：社稷（祭土神谷神）
- 卿大夫：五祀（祭门、户、中溜、灶、行）

## 5. 经济制度

### 5.1 货币体系

**贝币**：
- 最早货币
- 天然海贝
- 1贝 = 基本单位

**铜币**：
- 布币（铲形）
- 刀币（刀形）
- 圜钱（圆形）
- 1钱 = 10贝

**黄金**：
- 1金 = 1000钱
- 用于大宗交易
- 赏赐、贿赂

### 5.2 税收制度

**井田制时期**：
- 公田劳役：无偿耕种公田
- 土地税：无（公田即税）
- 贡赋：土特产

**初税亩后**：
- 土地税：按亩征收（10-20%）
- 户税：按户征收
- 关市税：商业税

---

## 6. 军事制度

### 6.1 兵役制度

**国人当兵**：
- 士：必须当兵（贵族义务）
- 国人：平时务农，战时当兵
- 野人：不当兵，仅负责后勤

**军队编制**：
- 5人为伍
- 5伍为两（25人）
- 4两为卒（100人）
- 5卒为旅（500人）
- 5旅为师（2500人）
- 5师为军（12500人）

### 6.2 武器装备

**进攻武器**：
- 戈、矛、戟、剑
- 弓箭、弩
- 战车

**防护装备**：
- 盾、甲
- 头盔

**攻城器械**：
- 云梯、冲车
- 抛石机

- 五等爵位
- 井田制形态
