# 《采邑春秋》美术音效需求

## 版本信息
- **版本**：v1.0
- **最后更新**：2026-01-28
- **美术风格**：2D像素艺术 / 中国水墨画融合

## 1. 美术需求

### 1.1 整体美术风格

#### 艺术方向

**核心理念**："青铜器美学 + 水墨意境 + 像素精致"

**视觉参考**：
- 传统青铜器纹样（饕餮纹、云雷纹）
- 水墨山水画（宋代风格）
- 像素艺术（现代独立游戏）
- 敦煌壁画色彩

**配色方案**：

| 色系 | 主色 | 辅色 | 点缀色 |
|------|------|------|--------|
| 土地 | #8B7355 | #6B5345 | #AB9375 |
| 植物 | #7CBA5F | #5A9A3E | #9AD47F |
| 水域 | #4A90E2 | #3A70D2 | #6AB0F2 |
| 建筑 | #D4A574 | #B48554 | #F4C594 |
| 青铜 | #CD7F32 | #A06020 | #E89F42 |
| 丝绸 | #E6B800 | #C69700 | #FFD700 |

#### 分辨率与比例

**基础分辨率**：640×360

**像素密度**：
- 角色：32×32像素
- 建筑：64×64像素
- Tile：16×16像素

**缩放模式**：整数缩放（2x, 3x, 4x）

| 大夫 | 锦绣 | 紫色 | 金簪 | 贵气 |

## 2. 动画需求

### 2.1 角色动画

#### 玩家动画

**待机动画**：
- 帧数：4帧
- 时长：1秒循环
- 动作：轻微上下浮动 + 呼吸

**行走动画**：
- 帧数：8帧
- 时长：0.8秒循环
- 动作：左右腿交替 + 手臂摆动

**耕作动画**：
- 帧数：6帧
- 时长：1秒循环
- 动作：举起工具 → 挥下 → 收回

**战斗动画**：
- 普通攻击：4帧
- 蓄力攻击：8帧
- 受击：3帧
- 死亡：6帧

#### 门客动画

**工作动画**：
- 种田：弯腰 → 播种 → 站起
- 修缮：锤击 → 擦汗 → 锤击
- 守卫：站立 → 巡视 → 停留

**休闲动画**：
- 坐下：4帧
- 聊天：6帧（嘴巴动）
- 吃饭：8帧（咀嚼）

### 2.2 环境动画

#### 水面动画

**河流**：
- 纹理滚动（平移）
- 波光闪烁（透明度变化）
- 时长：2秒循环

**湖泊**：
- 微波荡漾（缩放）
- 反光效果（闪烁）

#### 植物动画

**树木**：
- 风吹摇摆（旋转2-3度）
- 时长：3秒循环

**作物**：
- 生长动画（从小到大）
- 摇曳（风吹）

**花朵**：
- 开放（缩放）
- 飘落（粒子）

#### 天气动画

**下雨**：
- 雨滴粒子（下落）
- 地面积水（透明层）
- 时长：持续

**下雪**：
- 雪花粒子（飘落+旋转）
- 地面积雪（白色层）
- 时长：持续

**落叶**：
- 叶子粒子（飘落+旋转）
- 时长：5秒

### 2.3 特效动画

#### 战斗特效

**攻击特效**：
- 刀光：白色弧形 + 渐隐
- 箭矢：快速移动 + 拖尾
- 冲击波：圆形扩散 + 渐隐

**伤害数字**：
- 上浮 + 缩放 + 渐隐
- 颜色：白色（普通）、黄色（暴击）、红色（致命）

#### 成功特效

**升级**：
- 金光闪耀
- 星星粒子（放射）
- 音效提示

**成就解锁**：
- 徽章弹出
- 光芒四射
- 粒子庆祝

## 3. 音效需求

### 3.1 音乐需求

#### 背景音乐（BGM）

**主旋律**：
- 风格：中国传统音乐（古琴、编钟、笛子）
- 节奏：缓慢、宁静
- 长度：3-5分钟循环
- 情绪：平和、悠扬

**变奏版本**：

| 版本 | 乐器 | 节奏 | 情绪 | 场景 |
|------|------|------|------|------|
| 早晨 | 笛子 + 琴 | 中速 | 清新 | 日常 |
| 正午 | 编钟 + 鼓 | 稍快 | 活力 | 工作 |
| 黄昏 | 古琴 solo | 缓慢 | 宁静 | 休息 |
| 深夜 | 箫 solo | 极慢 | 寂静 | 夜晚 |
| 战斗 | 鼓 + 号角 | 快 | 紧张 | 战斗 |
| 事件 | 全乐器 | 变化 | 戏剧性 | 剧情 |

**主题音乐**：
- 标题画面：恢弘大气（编钟+鼓）
- 成就：欢快喜庆（笛子+锣）
- 悲伤：哀婉动人（二胡）
- 胜利：激昂慷慨（全乐器）

#### 小游戏音乐

**编钟音游**：
- 编钟音阶（12音）
- 节奏伴奏
- 成功/失败音效

**养蚕温控**：
- 轻柔背景
- 温度警报音

## 4. 技术需求

### 4.1 美术资源格式

**图片格式**：
- PNG（透明）
- WebP（压缩）
- JPG（不透明大图）

**分辨率**：
- Sprite: 32×32, 64×64
- Tile: 16×16
- UI: 按需

**图集**：
- 使用TexturePacker打包
- 减少Draw Call

### 4.2 音频资源格式

**音频格式**：
- 音乐：OGG（高质量压缩）
- 音效：OGG（短音效）
- 语音：OGG

**采样率**：
- 音乐：44.1kHz
- 音效：22.05kHz
- 语音：44.1kHz

**比特率**：
- 音乐：128-192 kbps
- 音效：64-96 kbps
- 语音：96-128 kbps

### 4.3 优化策略

**美术优化**：
- 使用Tile Atlas
- 共用材质
- LOD（远距离低精度）

**音频优化**：
- 流式播放BGM
- 音效池管理
- 按需加载
