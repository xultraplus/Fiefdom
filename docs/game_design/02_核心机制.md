## 1. 井田制系统

### 1.1 基础机制

**核心概念**：每个田地单元为 **3x3 九宫格**，称为一个"井田"。

```
┌─────┬─────┬─────┐
│ 私 │ 私 │ 私 │
├─────┼─────┼─────┤
│ 私 │公田│ 私 │
├─────┼─────┼─────┤
│ 私 │ 私 │ 私 │
└─────┴─────┴─────┘
```

#### 公田（中央格）
- **产出归属**：强制上缴国君
- **玩家收益**：声望 + 国君赏赐（稀有物品）
- **维护要求**：必须投入劳动力，否则扣除声望
- **惩罚机制**：
  - 维护差 → 声望 -10/天
  - 声望过低 → "削爵"（爵位下降）

#### 私田（周围8格）
- **产出归属**：玩家所有
- **经济收益**：金钱 + 粮食
- **种植自由**：可种植五谷、桑树等

### 1.2 田地开垦系统

#### 开垦条件
| 条件类型 | 要求 |
|---------|------|
| 爵位限制 | 当前井田数 < 爵位上限 |
| 地形限制 | 仅限平原地形 |
| 空间要求 | 完整3x3区域（无障碍） |
| 资源消耗 | 铜钱 + 粮食 + 3个门客 |
| 间隔限制 | 井田之间至少间隔1格 |

#### 开垦成本
```gdscript
# 资源消耗公式
铜钱消耗 = 100 × 当前爵位等级
粮食消耗 = 50 × 当前爵位等级
门客工时 = 3个门客 × 4天
```

#### 开垦流程（5天）

**第1天：清除植被**
- 门客工作：清除杂草、石块
- 地形变化：荒地 → 净地
- 视觉表现：去除枯黄植被

**第2-3天：翻土**
- 门客工作：翻松土壤
- 地形变化：净地 → 松土
- 视觉表现：深褐色、耕作痕迹

**第4天：标记井田**
- 系统自动：标记中央为公田，周围为私田
- 地形变化：松土 → 可耕地
- 视觉表现：公田显示金色边框

**第5天：可种植**
- 玩家可以开始种植作物
- 地形锁定：该3x3区域永久为井田

#### 爵位与井田上限

| 爵位 | 等级 | 井田数 | 总格数 | 私田 | 公田 | 解锁条件 |
|------|------|--------|--------|------|------|----------|
| 下士 | 1 | 1个 | 9格 | 8格 | 1格 | 游戏开始 |
| 中士 | 2 | 2个 | 36格 | 32格 | 2格 | 声望100 + 周礼"中" |
| 上士 | 3 | 4个 | 144格 | 128格 | 4格 | 声望500 + 六艺×1达Lv.5 |
| 大夫 | 4 | 9个 | 324格 | 288格 | 9格 | 声望2000 + 完成初税亩选择 |
| 卿 | 5 | 16个 | 576格 | 512格 | 16格 | 声望10000 + 六艺均达Lv.10 |

### 1.3 作物系统

#### 五谷作物

| 作物 | 名称 | 生长天数 | 单价 | 水需求 | 特殊属性 |
|------|------|----------|------|--------|----------|
| 黍 | 黄米 | 30天 | 5铜 | 低 | 耐旱 |
| 稷 | 粟米 | 25天 | 4铜 | 低 | 适应性强 |
| 麦 | 小麦 | 40天 | 8铜 | 中 | 高价值 |
| 菽 | 大豆 | 35天 | 6铜 | 中 | 固氮（恢复地力） |
| 稻 | 水稻 | 50天 | 12铜 | 高 | 需水田 |

#### 桑蚕产业链

**阶段1：种桑**
- 生长周期：60天
- 产叶量：每株桑树每天10片桑叶
- 养蚕需求：1张蚕种需100片桑叶/天

**阶段2：养蚕（小游戏）**
- 玩法：温度控制（QTE快速点击）
- 最佳温度：22-26°C
- 温度过低：蚕生长缓慢
- 温度过高：蚕生病死亡
- 周期：20天

**阶段3：缫丝**
- 产出：1个蚕茧 → 5两生丝
- 工具：缫丝车（需解锁）
- 门客工作：墨家门客效率+30%

**阶段4：织绸**
- 产出：10两生丝 → 1匹丝绸
- 工具：织布机（需解锁）
- 价值：丝绸 = 高级货币，用于外交/贿赂

### 1.4 耕作循环

```
春分（播种）→ 谷雨（浇水）→ 小满（生长）→ 芒种（除草）→
夏至（生长）→ 大暑（浇水）→ 立秋（收获）→ 秋分（纳贡）
```

**节气加成**：
- 顺应节气种植：产量 +20%
- 违背节气种植：产量 -30%，可能歉收

## 2. 爵位晋升系统

### 2.1 五等爵位

| 爵位 | 英文 | 服饰 | 车马 | 住所 | 社交地位 |
|------|------|------|------|------|----------|
| 下士 | Lower Scholar | 布衣 | 牛车 | 茅屋3间 | 村落级别 |
| 中士 | Middle Scholar | 麻衣 | 牛车+彩绘 | 茅屋5间 | 乡镇级别 |
| 上士 | Upper Scholar | 丝绸 | 马车 | 瓦房7间 | 县城级别 |
| 大夫 | Grand Master | 锦绣 | 双马战车 | 庭院9间 | 诸侯级别 |
| 卿 | Minister | 貂裘 | 四马战车 | 城堡 | 朝廷级别 |

### 2.2 晋升条件

#### 下士 → 中士
- **声望**：≥100
- **周礼评分**："中"或以上
- **公田贡献**：累计产出 ≥100单位
- **六艺要求**：无

#### 中士 → 上士
- **声望**：≥500
- **周礼评分**："良"或以上
- **公田贡献**：累计产出 ≥500单位
- **六艺要求**：任意1种达到Lv.5

#### 上士 → 大夫
- **声望**：≥2000
- **周礼评分**："优"或以上
- **公田贡献**：累计产出 ≥2000单位
- **六艺要求**：任意2种达到Lv.8
- **特殊条件**：完成"初税亩"选择

#### 大夫 → 卿
- **声望**：≥10000
- **周礼评分**："上上"
- **公田贡献**：累计产出 ≥10000单位
- **六艺要求**：全部6种达到Lv.10
- **特殊条件**：周天子召见事件

### 2.3 周礼评分系统

**评分维度**（各占20%）：
1. **公田维护**：公田产出占应缴比例
2. **祭祀礼仪**：是否按时祭祀（春社、秋社）
3. **服饰规范**：是否穿着符合爵位的服饰
4. **赋税缴纳**：是否按时缴纳赋税
5. **门客供养**：门客数量是否符合爵位标准

**评分等级**：
- 上上（90-100分）：卿候选人
- 上中（80-89分）：晋升加成
- 上下（70-79分）：正常晋升
- 中上（60-69分）：晋升缓慢
- 中中（50-59分）：无法晋升
- 下下（0-49分）：削爵风险

## 3. 六艺系统

### 3.1 礼 (Rites)

**功能**：
- 爵位晋升的必要条件
- 社交互动成功率
- 避免被霸主讨伐

**升级方式**：
- 祭祀活动（+10 XP）
- 宴请宾客（+5 XP/人）
- 着装规范（+2 XP/天）

**解锁内容**：
| 等级 | 解锁 |
|------|------|
| Lv.1 | 基础社交 |
| Lv.3 | 乡镇级宴会 |
| Lv.5 | 县城级外交 |
| Lv.8 | 诸侯级朝见 |
| Lv.10 | 周天子召见 |

**玩法机制**：
- **着装小游戏**：根据场合选择正确服饰
- **礼仪QTE**：按正确顺序行礼
- **祭祀仪式**：在正确时间使用正确祭品

### 3.2 乐 (Music)

**功能**：
- 范围Buff（提升门客效率）
- 安抚民心（降低犯罪率）
- 宴会必备

**升级方式**：
- 编钟演奏（音游小游戏）
- 铸造青铜乐器（+15 XP）
- 欣赏宫廷乐（+5 XP）

**音游机制**：
- 节奏条：音符下落时按对应键
- 难度：Lv.1-3简单，Lv.4-7中等，Lv.8-10复杂
- 乐器：编钟（12音）、磬、鼓

**Buff效果**：
| 等级 | 效果 | 范围 |
|------|------|------|
| Lv.1-3 | 工作效率+5% | 1个井田 |
| Lv.4-6 | 工作效率+10% | 所有井田 |
| Lv.7-9 | 工作效率+15% + 忠诚度+0.5/天 | 所有井田 |
| Lv.10 | 工作效率+20% + 忠诚度+1/天 | 所有井田 + 周边村落 |

### 3.3 射 (Archery)

**功能**：
- 狩猎获取稀有皮毛
- 塔防模式：射杀入侵野兽
- 战斗系统主技能

**升级方式**：
- 狩猎（+10 XP/次）
- 靶场练习（小游戏）
- 战斗（+20 XP/击杀）

**战斗机制**：
- **射击模式**：蓄力弓箭（按住蓄力，松开射击）
- **箭支类型**：
  - 普通箭：基础伤害
  - 火箭：燃烧效果
  - 毒箭：持续伤害
  - 穿甲箭：破甲

| 等级 | 伤害 | 射程 | 特殊 |
|------|------|------|------|
| Lv.1 | 10 | 3格 | 无 |
| Lv.5 | 25 | 5格 | 解锁火箭 |
| Lv.8 | 40 | 7格 | 解锁毒箭 |
| Lv.10 | 60 | 10格 | 解锁穿甲箭 |

### 3.4 御 (Chariot)

**功能**：
- 战车驾驶（跑图）
- 战车冲锋（群杀技能）
- 背包负重增加

**升级方式**：
- 驾车出行（+2 XP/格）
- 战车训练（小游戏）
- 战斗冲锋（+30 XP/次）

**战车系统**：
| 等级 | 战车类型 | 速度 | 冲击伤害 | 负重 |
|------|----------|------|----------|------|
| Lv.1 | 牛车 | 慢 | 0 | 20格 |
| Lv.5 | 单马战车 | 中 | 30 | 50格 |
| Lv.8 | 双马战车 | 快 | 60 | 100格 |
| Lv.10 | 四马战车 | 极快 | 100 | 200格 |

**驾驶小游戏**：
- 方向键控制战车
- 躲避障碍
- 冲撞敌人

### 3.5 书 (Calligraphy)

**功能**：
- 科技树解锁
- 辩论系统（舌战）
- 典籍编纂

**升级方式**：
- 夜读竹简（+5 XP/卷）
- 修复古籍（+15 XP）
- 辩论胜利（+20 XP）

**科技树**：
| 科技 | 前置 | 效果 |
|------|------|------|
| 铁犁 | 书Lv.3 | 耕作效率+50% |
| 牛耕 | 书Lv.5 | 耕作速度+100% |
| 水利 | 书Lv.7 | 自动浇水 |
| 轮作 | 书Lv.8 | 地力不下降 |
| 桑蚕 | 书Lv.2 | 解锁桑蚕产业链 |

**辩论系统**：
- **回合制**：选择论点反击
- **属性**：使用"书"等级判定胜负
- **奖励**：稀有书籍、门客、声望

### 3.6 数 (Math)

**功能**：
- 交易系统（砍价）
- 账目管理（优化门客AI）
- 税率优化

**升级方式**：
- 商业交易（+1 XP/10铜）
- 核算账目（小游戏）
- 优化管理（+10 XP/次）

**交易砍价**：
- 玩法：数字拼图
- 难度：Lv.1-3简单（2-3个数字），Lv.4-7中等（4-5个），Lv.8-10复杂（6+个）
- 效果：购买折扣-5% ~ -30%

**账目管理**：
- 平衡门客工作分配
- 优化产出效率
- 降低粮食浪费

- 落魄贵族：70-90

**忠诚度变化**：
| 事件 | 变化 |
|------|------|
| 每日食宿良好 | +2 |
| 每日食宿恶劣 | -5 |
| 发放奖金 | +10 |

| 敌对学派在场 | -2/人 |

- ≥80：忠诚，不会离开
- 50-79：稳定，可能被高薪挖走
- 30-49：动摇，可能潜逃

### 4.4 工作分配

#### 工作类型

| 工作 | 消耗 | 产出 | 推荐学派 |
|------|------|------|----------|
| 种田 | 1粮食/天 | 作物 | 农家 |
| 修缮 | 0 | 设施耐久+10 | 墨家 |
| 公田 | 1粮食/天 | 声望+5 | 法家、儒家 |
| 守卫 | 2粮食/天 | 安全+5 | 兵家 |
| 研习 | 1书籍/天 | 科技点 | 儒家、墨家 |

#### 工作效率公式

```
基础效率 = 10 单位/天
学派加成 = 基础效率 × 学派特质
等级加成 = 基础效率 × (忠诚度/100)
总效率 = 基础效率 × (1 + 学派加成) × (忠诚度/100)
```

## 5. 历史事件系统

### 5.1 初税亩改革事件

**触发条件**：
- 游戏进行到第3年（或爵位达"大夫"）
- 公田产出 ≥ 总产出的30%
- 未触发过此事件

**事件流程**：

#### 阶段1：国君密诏
- **使者来访**："周边诸侯已行初税亩，君上意下如何？"
- **玩家选择**：

**选项A：支持改革（法家路线）**
- 所有公田变私田
- 固定20%税率（可降至15%）
- 金钱+300%
- 周礼评分永久"下下"
- 儒家门客忠诚-50
- 法家招募费-50%

**选项B：坚守传统（儒家路线）**
- 保持井田制
- 周礼评分"上上"
- 儒家门客忠诚+30
- 每年春秋获国君赏赐
- 解锁"周天子召见"
- 法家门客无法招募

**选项C：拖延观望**
- 推迟1年
- 每月100铜钱"研究经费"
- 1年后强制选A或B

### 5.2 其他重大事件

| 事件 | 触发条件 | 选择 | 后果 |
|------|----------|------|------|
| 孔子来访 | 儒家路线+礼≥Lv.8 | 接受/拒绝 | 解锁六艺加成/无 |
| 三家分晋 | 第5年 | 支持晋/魏/赵/韩 | 外交关系变化 |
| 蛮族入侵 | 随机 | 抵降/防守/迁徙 | 战斗或损失 |
| 周天子召见 | 卿爵位+周礼"上上" | 朝见/拒绝 | 成为王朝重臣/失去机会 |
| 蝗灾 | 随机 | 祭祀/捕打/忍受 | 不同的粮食损失 |
