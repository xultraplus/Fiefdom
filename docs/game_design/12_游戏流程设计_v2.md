# 《采邑春秋》游戏流程设计

## 版本信息
- **版本**：v2.0
- **最后更新**：2026-01-28
- **目的**：详细描述玩家在游戏中的完整体验流程

---

## 1. 整体流程

### 1.1 游戏结构

```
开始界面
  ↓
角色创建
  ↓
教学关卡
  ↓
主游戏循环
  ↓
结局
  ↓
结算界面
  ↓
新游戏+ / 主菜单
```

### 1.2 时间结构

**短期**：单日活动（卯时→亥时）
**中期**：季度循环（春→夏→秋→冬）
**长期**：爵位晋升（下士→卿）

### 2.3 教学关卡

#### 教学目标

**目标1：基础移动**
- 教学内容：WASD移动
- 完成条件：走到指定位置
- 奖励：无

**目标2：种植作物**
- 教学内容：选择种子、种植
- 完成条件：种植1格作物
- 奖励：+10 XP

**目标3：浇水**
- 教学内容：使用水壶浇水
- 完成条件：浇水1次
- 奖励：+5 XP

**目标4：收获**
- 教学内容：收获成熟作物
- 完成条件：收获1次
- 奖励：+15 XP

**目标5：分配门客**
- 教学内容：打开管理面板、分配工作
- 完成条件：分配1个门客
- 奖励：+20 XP

**目标6：结束一天**
- 教学内容：使用睡眠功能
- 完成条件：睡觉1次
- 奖励：无

## 3. 核心循环

### 3.1 单日活动

#### 卯时（05:00-07:00）规划阶段

**玩家操作**：
1. 查看顶部信息栏（声望、爵位、资源）
2. 检查天气、节气
3. 打开管理面板，分配门客任务
4. 查看事件日志

**系统处理**：
- 计算夜间事件结果
- 更新门客状态（疲劳度、忠诚度）
- 显示随机事件（如有）

**UI界面**：
```
┌──────────────────────────────────────┐
│  📅 卯时（05:00）                     │
│  🌤️ 天气: 晴朗                       │
│  📜 节气: 春分 (播种季节)              │
│  💡 提示: 适合种植作物，产量+20%       │
│                                      │
│  [分配门客] [查看日历] [关闭]         │
└──────────────────────────────────────┘
```

#### 辰-申时（07:00-15:00）执行阶段

**玩家操作**：
1. 移动到井田
2. 种植/浇水/收获作物
3. 探索周边区域
4. 与访客交互

**门客行为**：
- 根据分配自动工作
- 工作期间无需玩家操作
- 完成后报告结果

**时间推进**：
- 每个操作消耗0.1-0.5天
- 自动记录到时间轴

#### 酉-亥时（17:00-23:00）修养阶段

**玩家操作**：
1. 夜读六艺（获得XP）
2. 社交宴请（提升忠诚度）
3. 处理突发事件
4. 准备睡觉

**可选活动**：
```
┌──────────────────────────────────────┐
│  🌙 今晚做什么？                      │
├──────────────────────────────────────┤
│  [📖 夜读六艺] (+10 XP)              │
│  [🍷 宴请门客] (忠诚度+5)            │
│  [🎵 演奏编钟] (声望+10)              │
│  [💤 直接睡觉]                        │
└──────────────────────────────────────┘
```

### 3.2 季节循环

#### 春季（春分-谷雨）

**季节特色**：
- 播种季节：作物产量+20%
- 森林果实丰富
- 动物繁殖期

**推荐活动**：
- 大量种植作物
- 采集森林果实
- 招募新门客

**特殊事件**：
- 春社祭祀（声望+50）
- 动物繁殖（家畜+1）

#### 夏季（立夏-大暑）

**季节特色**：
- 生长季节：作物生长+30%
- 浇水需求高
- 可能干旱

**推荐活动**：
- 勤浇水
- 准备抗旱
- 防范蛮族

**特殊事件**：
- 端午龙舟（门客忠诚度+10）
- 干旱灾害（作物损失）

#### 秋季（立秋-霜降）

**季节特色**：
- 收获季节：作物产量+10%
- 粮食价格下降20%
- 动物迁徙

**推荐活动**：
- 大量收获
- 出售粮食
- 狩猎动物

**特殊事件**：
- 秋社祭祀（声望+50）
- 蛮族入侵（战斗）

#### 冬季（立冬-大寒）

**季节特色**：
- 休耕季节：无法种植
- 粮食价格上涨30%
- 燃料消耗增加

**推荐活动**：
- 夜读六艺
- 社交宴请
- 制作工具

**特殊事件**：
- 腊八祭祖（声望+30）
- 暴雪灾害（无法出门）

## 4. 晋升流程

### 4.1 爵位晋升条件检查

**检查时机**：
- 每季度结束时
- 完成重大事件后

**检查内容**：
1. 声望是否达标
2. 周礼评分是否达标
3. 公田贡献是否达标
4. 六艺等级是否达标

**晋升对话框**：
```
┌──────────────────────────────────────┐
│  🎉 恭喜晋升！                        │
├──────────────────────────────────────┤
│  您已达到晋升条件！                   │
│                                      │
│  当前爵位: 下士                       │
│  晋升爵位: 中士                       │
│                                      │
│  新解锁内容:                          │
│  ✓ 井田数量: 1→2                     │
│  ✓ 新服饰: 麻衣                      │
│  ✓ 新门客上限: 10→15                 │
│                                      │
│  [查看详情] [接受晋升]               │
└──────────────────────────────────────┘
```

### 4.2 晋升仪式

**仪式内容**：
1. 前往王都（或本县城）
2. 朝见上级贵族
3. 行礼（QTE小游戏）
4. 接受册封
5. 获得赏赐

**仪式动画**：
- 行礼动画
- 册封动画
- 赏赐展示

**奖励内容**：
- 新服饰
- 新井田
- 声望+100
- 稀有礼物

## 5. 重大事件流程

### 5.1 初税亩改革事件

#### 触发流程

**第1阶段：铺垫（前3个月）**
- 随机访客提到"初税亩"
- 商人谈论改革影响
- 门客表达看法（学派不同）

**第2阶段：触发（第3年或爵位达大夫）**
- 国君使者来访
- 触发事件对话框

**第3阶段：选择**
- 玩家做出选择
- 系统记录选择
- 应用立即效果

**第4阶段：长期影响**
- 后续事件变化
- 门客忠诚度变化
- 经济模式改变

#### 事件详情

**使者来访**：
```
┌──────────────────────────────────────┐
│  💬 国君密诏                         │
├──────────────────────────────────────┤
│  ┌─────────────────────────────────┐ │
│  │  [使者立像]                     │ │
│  └─────────────────────────────────┘ │
│                                      │
│  使者："卿家，周边诸侯已行初税亩，     │
│  国君有意效仿，不知卿以为如何？"      │
│                                      │
│  [选项详细内容...]                    │
│                                      │
│  [A: 支持改革] [B: 坚守传统] [C: 拖延]│
└──────────────────────────────────────┘
```

**后续变化**：
- 支持改革：公田全变私田，税率20%
- 坚守传统：井田制不变，每年赏赐
- 拖延观望：每月100铜，1年后强制选

### 5.2 孔子来访事件

**触发条件**：
- 儒家路线
- "礼"≥Lv.8
- 未触发过

**事件流程**：
1. 孔子造访封地
2. 与孔子对话（选择选项）
3. 孔子考校六艺
4. 获得奖励

**奖励**：
- 孔子弟子加入（儒家门客，忠诚度100）
- 稀有典籍
- 声望+500

## 6. 战斗流程

### 6.1 遭遇敌人

**触发方式**：
- 探索时随机遭遇
- 蛮族入侵封地
- 主动攻击兽穴

**遭遇界面**：
```

│  ⚔️ 遭遇敌人！                        │
├──────────────────────────────────────┤
│  ┌─────────────────────────────────┐ │
│  │  [蛮族战士×3]                   │ │
│  │  生命: 100/100                  │ │
│  └─────────────────────────────────┘ │
│                                      │
│  [战斗] [逃跑] [观察]               │
└──────────────────────────────────────┘
```

### 6.2 战斗过程

**回合结构**：
1. 抽牌阶段：抽5张战斗卡
2. 行动阶段：打出卡牌（受行动点限制）
3. 敌人阶段：敌人攻击
4. 结束阶段：弃置所有手牌

**胜利条件**：
- 所有敌人生命值归零

**失败条件**：
- 玩家生命值归零

**战斗结束界面**：

## 7. 结局流程

### 7.1 结局类型

#### 声望胜利（儒家路线）

**触发条件**：
- 爵位达到"卿"
- 周礼评分"上上"
- 未触发其他结局

**结局动画**：
- 周天子召见
- 册封为"王朝重臣"
- 封地繁荣昌盛
- 传承后世

**结局文字**：
> "卿家复兴礼乐，功在社稷。
> 朕特封卿为王朝重臣，子孙永享荣光。"

#### 经济胜利（法家路线）

**触发条件**：
- 金钱达到10000金
- 支持初税亩改革

**结局动画**：
- 成为富可敌国的大商人
- 诸侯争相结交
- 封地商业繁荣
- 建立商业帝国

**结局文字**：
> "卿家富甲天下，连诸侯亦不敢轻视。
> 此乃经济之胜利也。"

#### 军事胜利（兵家路线）

**触发条件**：
- 统一周边5个小国
- 击退3次蛮族入侵

**结局动画**：
- 成为一方霸主
- 建立私人军队
- 威震四方
- 诸侯畏惧

**结局文字**：
> "卿家文武双全，威震四方。
> 今封卿为方伯，统领一方。"

#### 文化胜利（全六艺）

**触发条件**：
- 六艺全部达到Lv.10
- 门客中各学派平衡

**结局动画**：
- 编纂典籍
- 弟子满天下
- 成为诸子百家领袖
- 流芳百世

**结局文字**：
> "卿家六艺精通，德才兼备。
> 真乃当世君子，后世楷模。"
