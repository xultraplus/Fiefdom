# 《采邑春秋》元游戏系统

## 系统概述

元游戏（Meta-Game）是指游戏之外的系统，包括：
1. **存档系统**：保存/读取进度
2. **成就系统**：长期目标
3. **多周目系统**：新游戏+（New Game+）
4. **解锁系统**：内容解锁
5. **统计系统**：玩家数据追踪

---

## 1. 存档系统

### 1.1 存档机制

#### 自动存档

**触发条件**：
- 每天结束时（睡觉）
- 完成重大事件后
- 晋升爵位后

**存档数量**：3个（循环覆盖）
- 存档1：最新
- 存档2：倒数第二新
- 存档3：倒数第三新

#### 快速存档

**快捷键**：F5

**存档数量**：1个
- 覆盖上一次快速存档

**适用场景**：
- 玩家想手动保存当前进度
- 重要决策前保存

#### 手动存档

**入口**：暂停菜单 → 保存游戏

**存档数量**：10个
- 可命名
- 可覆盖
- 可删除

**存档信息**：
```
┌──────────────────────────┐
│  存档1: 春秋第三年        │
│  ────────────────         │
│  时间: 2026-01-28 15:30   │

│  爵位: 上士               │
│  门客: 12人               │
│  资金: 5,230铜             │
│                            │
│  [覆盖] [删除] [加载]      │
└──────────────────────────┘
```

- 保留最近5个备份

## 3. 新游戏+系统（New Game+）

### 3.1 继承机制

#### 可继承内容

| 内容 | 继承比例 | 说明 |
|------|----------|------|
| 六艺等级 | 20% | 最高继承Lv.2 |
| 成就解锁 | 100% | 全部保留 |
| 特殊称号 | 100% | 全部保留 |
| 特殊服饰 | 100% | 全部保留 |

#### 不可继承内容

- 金钱、粮食等资源
- 门客
- 建筑
- 爵位
- 事件进度

### 3.2 二周目特性

#### 新增内容

- **难度选择**：可选择更高难度
- **特殊模式**：
  - 无限制模式：资源无限
  - 困难模式：资源减半
  - 专家模式：永久死亡

#### 奖励内容

- **特殊成就**：仅二周目可解锁
- **隐藏内容**：
  - 隐藏门客
  - 隐藏事件
  - 隐藏结局
