# 《采邑春秋》游戏总览

## 基本信息

| 项目 | 内容 |
|------|------|
| **项目名称** | 采邑春秋 (Project: Fiefdom) |
| **游戏类型** | 模拟经营 / 角色扮演 / 历史策略 |
| **目标平台** | PC (Windows/Mac/Linux) |
| **开发引擎** | Godot 4.6 |
| **美术风格** | 2D像素艺术 / 手绘风格 |
| **视角** | 俯视角 (Top-down) |

## 一句话介绍

一款以中国春秋时期为背景的"士大夫模拟器"，玩家扮演没落贵族经营封地，通过井田制复兴经济、修习君子六艺、招募诸子百家门客，在礼崩乐坏的乱世中守护一方桃源。

## 核心体验支柱

### 1. 考究的国风种田
- 不仅是种菜，更是对"五谷、桑蚕、井田制"的系统模拟
- 历史真实的作物：黍、稷、稻、麦、菽
- 桑蚕产业链：种桑 → 养蚕 → 缫丝 → 织绸
- 二十四节气系统：顺应天时才能丰收

### 2. 士大夫的阶层跃迁
- 从"下士"到"卿"的五级爵位系统
- 体验等级森严的春秋贵族生活
- 六艺修习：礼、乐、射、御、书、数
- 声望管理：维持"周礼"评分以避免削爵

### 3. 门客管理与自动化
- 如同孟尝君般养士
- 招募诸子百家：儒家、墨家、法家、农家、兵家
- 每个学派有独特能力和性格
- 将繁琐劳动转化为人员管理策略

## 游戏特色

### 独特卖点 (USP)

1. **历史真实性**
   - 基于真实春秋历史（公元前770-476年）
   - 考据严谨的礼制、官制、物产
   - 历史事件：初税亩改革、三家分晋、孔子周游

2. **井田制核心机制**
   - 3x3九宫格布局：公田(1) + 私田(8)
   - 公私利益博弈：公田维持声望，私田获取收益
   - 后期重大选择：支持初税亩改革 vs 坚守井田传统

3. **六艺RPG系统**
   - 替代传统RPG属性(力量/敏捷/智力)
   - 每种艺术对应独特玩法：
     - 礼：社交、爵位晋升
     - 乐：音游、范围Buff
     - 射：狩猎、塔防射击
     - 御：战车驾驶、跑图
     - 书：科技树、辩论系统
     - 数：交易、账目管理

4. **门客学派系统**
   - 五大学派，不同能力与性格
   - 忠诚度管理：待遇差会潜逃或倒戈
   - 学派冲突：儒法对立、墨农搭配

5. **事件驱动叙事**
   - 随机事件：访客、灾难、机遇
   - 历史大事件：初税亩、周天子召见
   - 多分支选择，影响游戏走向

## 核心循环

### 秒级循环（操作层）
- 移动角色、挥舞农具、交互对象
- 战斗：射箭、战车冲撞
- 小游戏：编钟演奏、养蚕温控

### 日级循环（战术层）

**卯时（05:00-07:00）- 规划**
- 查看黄历/政令
- 分配门客任务
- 检查天气/节气

**辰-申时（07:00-15:00）- 执行**
- 亲耕私田（恢复体力）
- 督办公田（维持爵位）
- 外出探索（攘夷/采集）

**酉-亥时（17:00-23:00）- 修养**
- 夜读修习六艺
- 社交宴请
- 处理突发事件

### 长线循环（战略层）

**季度（节气）**
- 播种 → 浇水 → 收获
- 祭祀活动（春社、秋社）
- 纳贡上缴

**年度**
- 爵位晋升考核
- 科技解锁（铁器/牛耕）
- 扩大封地（开垦新井田）
- 家族联姻/外交

## 胜利条件

### 多重结局

1. **声望胜利（儒家路线）**
   - 达到"卿"爵位
   - 周礼评分"上上"
   - 被周天子召见

2. **经济胜利（法家路线）**
   - 积累财富10000金
   - 支持初税亩改革
   - 成为霸主盟友

3. **军事胜利（兵家路线）**
   - 统一周边5个小国
   - 击退3次蛮族入侵
   - 建立私人军队

4. **文化胜利（全六艺）**
   - 六艺全部达到Lv.10
   - 门客中各学派平衡
   - 编纂典籍流传后世

## 失败条件

1. **削爵**
   - 声望降至0
   - 公田产出长期不足
   - Game Over：贬为庶人

2. **灭国**
   - 封地被蛮族攻破
   - 无法偿还债务
   - Game Over：流亡他国

3. **叛变**
   - 门客集体叛逃
   - 失去所有劳动力
   - Game Over：封地荒废
