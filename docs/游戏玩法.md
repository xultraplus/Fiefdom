这是一个关于**《采邑春秋》（暂定名）**的核心游戏玩法说明文档（Game Mechanics Design）。

这份说明旨在明确玩家在游戏中的具体操作、核心循环以及如何体现“春秋士大夫”的独特体验。我们将玩法拆解为**基础循环**、**特色经济**、**角色养成**和**世界交互**四个维度。

---

### 一、 核心循环 (Core Loop)
**玩家每一天（约现实时间15-20分钟）的行动流程：**

1.  **晨起（卯时）**：查看**黄历/竹简**。
    *   确认今日节气（如“芒种”需忙耕作，“冬至”需祭祀）。
    *   查看国君发布的政令（如“急需征收兵器”或“需上缴粮食”）。
2.  **劳作与管理（辰时-申时）**：
    *   **亲耕**：在私田中耕种、浇水（消耗体力）。
    *   **督办**：指派“门客”去公田劳作，或去矿山采铜（消耗粮食库存）。
    *   **制艺**：制作陶器、青铜器、纺织丝绸。
3.  **社交与游历（酉时）**：
    *   去村口/驿站接待路过的游学士子（触发随机事件/辩论）。
    *   去邻国/郊外进行狩猎或采集。
4.  **夜读与修整（戌时-亥时）**：
    *   点灯夜读（消耗灯油，提升“智略”或“礼学”经验）。
    *   安排明日门客的工作。
5.  **结算（入寝）**：
    *   结算当日体力和健康恢复。
    *   结算门客的忠诚度变化（如果今天没给够饭吃，门客会逃跑）。

---

### 二、 特色经济系统：井田与纳贡

这是本作区别于《星露谷》的最大亮点，强调**“公私分明”**。

#### 1. 井田制 (The Well-Field System)
地图上的耕地默认呈“井”字形分布（3x3九宫格）。
*   **私田（周围8格）**：
    *   产出归玩家所有。
    *   用于：食用（恢复体力）、交易（换取铜币）、供养门客。
*   **公田（中间1格）**：
    *   产出不可拿走，直接存入“公仓”。
    *   **玩法逻辑**：玩家必须定期维护公田。如果公田荒芜，**“爵位”**会下降，甚至被褫夺封地（Game Over）；如果公田产量高，国君会赏赐稀有种子或青铜器。
*   **后期演变**：
    *   通过解锁科技（如“牛耕”）或政策（如“初税亩”），可以打破井田限制，将公田私有化，但这会大幅降低你的“周礼”评价，导致儒家NPC远离你，但法家NPC会支持你。

#### 2. 六畜与桑蚕
*   **六畜**：马、牛、羊、鸡、犬、豕。
    *   **祭祀功能**：不同于普通养殖，春秋时期牛/羊/豕主要用于**祭祀**。祭祀等级越高，获得的Buff越强（如“风调雨顺”减少旱灾）。
*   **桑蚕纺织**：
    *   种植桑树 -> 养蚕（需要控制温度） -> 缫丝 -> 织布。
    *   丝绸是游戏中的**硬通货**，比铜币更适合用于高级外交和贿赂。

### 三、 角色养成：六艺技能树

玩家没有通用的“等级”，而是修习**“君子六艺”**。

| 技能 (Arts) | 升级方式 | 解锁内容 |
| :--- | :--- | :--- |
| **礼 (Rites)** | 祭祀、举办宴席、送礼 | 提升爵位上限，招募高级名士，避免被其他诸侯讨伐。 |
| **乐 (Music)** | 敲击编钟（音游小游戏）、采集矿石铸钟 | 提升区域Buff范围，安抚暴动的流民，增加门客忠诚。 |
| **射 (Archery)** | 狩猎、射箭小游戏 | 提升远程攻击力，解锁高级弓弩，提升狩猎稀有皮毛概率。 |
| **御 (Chariot)** | 驾驶马车跑图、赛马 | 提升大地图移动速度，增加背包负重，解锁战车冲撞技能。 |
| **书 (Literacy)** | 夜读、修复竹简、与学者辩论 | 解锁更高级的合成配方（如水利设施），提升辩论胜率。 |
| **数 (Math)** | 交易、丈量土地、管理账目 | 减少纳税额度（精算），提升商店售卖价格，提升门客工作效率。 |

### 四、 门客与管理系统 (Retainer Management)

你不能一个人干完所有活，你需要像孟尝君一样**“养士”**。

1.  **招募**：
    *   流民（低级）：消耗少量粮食，只能干粗活（种地、砍树）。
    *   士人（中级）：需要特定声望，可以协助管理（自动收税、自动修补围墙）。
    *   名家（高级）：如孙武、墨子等。需要完成特定剧情任务链。
2.  **性格与派系**：
    *   如果你的封地主要推行“法家”（严刑峻法，高税收高产出），儒家门客的忠诚度会每日下降。
    *   你需要平衡不同派系的门客，或者专精一派。
3.  **自动化**：
    *   在“竹简管理界面”，将门客拖拽到特定地块。
    *   例如：将“墨家弟子”拖到农田，他会制造水车自动浇水；拖到工坊，他会加速青铜剑的生产。

### 五、 战斗与探险：攘夷 (Expedition)

战斗不是为了杀怪升级，而是为了**“保境安民”**和**“获取稀有资源”**。

1.  **战斗模式**：
    *   2D俯视角即时战斗（类似萨尔达传说：缩小帽）。
    *   特色机制：**战车冲锋**。在开阔地带，玩家可以召唤战车，此时移动速度极快，靠撞击造成高伤，但转向困难（适合对付成群的蛮族步兵）。
2.  **敌人类型**：
    *   **野人/盗匪**：偷窃你的仓库，破坏农田。需定期清理周边的据点。
    *   **山戎/狄人**：装备精良的入侵者，会发动攻城战（塔防玩法）。
    *   **神兽/猛兽**：虎、熊、甚至传说中的夔牛（Boss战），掉落制作“礼器”的核心材料。
3.  **死亡惩罚**：
    *   如果战败，不会立刻死亡，而是被俘虏。需要花费赎金（铜币或丝绸）赎回自己，或者等待门客来救你。这非常符合春秋时期的贵族战争规则。

---

### 六、 胜利条件与结局 (Endgame)

游戏没有唯一的终点，根据玩家倾向有不同结局：

1.  **霸主结局**：武力征服周边盗匪，兵强马壮，协助你的国君成为“春秋五霸”之一。
2.  **圣人结局**：六艺全满，门客中有大量学者，著书立说，成为后世敬仰的诸子百家之一。
3.  **陶朱公结局**：积累巨额财富，垄断丝绸和盐铁贸易，逍遥于江湖。
4.  **隐士结局**：拒绝所有战争和政治，将封地建设成绝对的世外桃源（桃花源记）。

### 给开发者的 Godot 实现提示：

*   **井田制实现**：在 `TileMapLayer` 中使用 Custom Data Layer 标记每个格子的属性（Owner: Player 或 Owner: King）。收获逻辑中检查这个属性即可。
*   **六艺系统**：建立一个全局 `Resource` 或 `Dictionary` 存储 6 个变量。UI 界面直接绑定这些变量。
*   **门客AI**：使用 Godot 的 `NavigationAgent2D`。给门客一个状态机：`Idle` -> `CheckTask` -> `MoveToTarget` -> `Work (Play Animation)` -> `Return`。
